# Supex

## Description

Supex is a smart contract that allows users to create tickets for charging their gadgets.

## Features

- Create tickets for charging gadgets
- Update ticket status
- Cancel tickets
- Process payments
- Release payments
- Refund payments
- Check charging status
- Get estimated completion
- Pause and unpause the contract
- Batch update status

## Usage

sncast declare --contract-name Supex
sncast deploy --class-hash <class_hash>
